# 🚀 AI Resume Generator (LayoutGAN)

An intelligent web application that generates professional resumes using AI-based layout prediction. Built with **LayoutGAN, Streamlit, and ReportLab**, this project creates clean, structured, and ATS-friendly resumes instantly.

---

## 🌟 Features

✅ AI-based resume layout generation  
✅ Clean and professional PDF output  
✅ Profile photo support  
✅ Skills displayed in bullet format  
✅ Layout visualization using matplotlib  
✅ Web app interface using Streamlit  
✅ Downloadable resume (PDF)  

---

## 🧠 Tech Stack

- Python 🐍
- Streamlit 🌐
- LayoutGAN (concept-based)
- NumPy
- Matplotlib 📊
- ReportLab 📄

## ⚙️ How It Works

1. User enters details (name, summary, skills, etc.)
2. Layout is generated using AI-inspired logic
3. Resume is structured and formatted
4. PDF is generated using ReportLab
5. User downloads the resume instantly
